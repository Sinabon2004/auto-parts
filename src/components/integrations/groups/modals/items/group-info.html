Переменные:

  • Запчасть:
    • {{ part.article }}: Артикул
    • {{ part.fullname }}: Название
    • {{ part.oem }}: OEM
    • {{ part.oem_description }}: Описание ОЕМа
    • {{ part.description }}: Описание экземляра
    • {{ part.coalescent_description }}: Объединённое описание OEMа и запчасти
    • {{ part.condition }}: Состояние (used или new)
    • {{ part.markdowned }}: Уценено?
    • {{ part.feature }}: Особенность
      • {{ part.feature.name }}: Название
      • {{ part.feature.comment }}: Комментарий
    • {{ part.type.seo_keywords }}: Поисковый индекс по узлам.
    • {{ part.donor.vehicle_model_with_make }}: Модель и производитель донора
    • {{ part.donor.vehicle_model_with_make_and_years }}: Модель и производитель донора (с годами)
    • {{ part.applicability }}: Применимость в различных форматах:
      • {{ part.applicability.count }}: Количество моделей второго уровня в применимости.
      • {{ part.applicability.v1 }}: Модели первого уровня с производителями (через запятую). Заполняется если в применимости не более 5 моделей.
      • {{ part.applicability.v2 }}: Модели второго уровня с производителями и синонимами (уникальные, списком в курсиве). Заполняется если в применимости не более 40 моделей.
      • {{ part.applicability.v2a }}: Аналогично v2, но без синонимов.
      • {{ part.applicability.v3 }}: Только одна модель второго уровня с производителем и синонимами (в скобках).
      • {{ part.applicability.v3a }}: Аналогично v3, но без синонимов.
      • {{ part.applicability.v4 }}: Только производители (списком).
      • {{ part.applicability.v4a }}: Аналогично v4, но без синонимов.
  • Объявление:
    • {{ ad.uuid }}: UUID
    • {{ ad.source_ids }}: ID источников, на основе которых создано объявление.
